# python-fastapi-jwt
python fastapi jsonwebtoken

## clone 

```sh
git clone 
```

## install package

```sh
pip install -r req.txt
OR
pip install "fastapi[all]"
pip install "uvicorn[standard]"
pip install "python-jose[cryptography]" "passlib[bcrypt]" python-multipart
```

## run

```sh
uvicorn main:app --reload
```
